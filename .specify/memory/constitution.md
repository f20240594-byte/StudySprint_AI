# StudySprint AI Constitution

## Purpose

StudySprint AI helps students create personalized study schedules based on exam dates, preparation level, and subject priority.

## Principles

### Simplicity First

The system must remain beginner-friendly and easy to understand.

### User Focus

Every feature should improve student productivity.

### Explainability

Study-hour allocation must be explainable through priority, preparation, and exam urgency.

### Reliability

The backend should always return valid JSON responses.

### Maintainability

Code should be modular, documented, and easy to extend.

### Performance

Schedule generation should complete within a few seconds.
