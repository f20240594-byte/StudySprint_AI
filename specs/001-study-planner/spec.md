# Feature Specification

## Feature

Personalized Study Planner

## Problem

Students often struggle to distribute study time effectively across multiple subjects with different exam dates and preparation levels.

## User Stories

### US-1

As a student,

I want to enter subjects and exam dates,

So that I can generate a study plan.

### US-2

As a student,

I want to specify my preparation level,

So that weaker subjects receive more attention.

### US-3

As a student,

I want to specify priority levels,

So that important subjects receive more study time.

### US-4

As a student,

I want a day-wise study schedule,

So that I know exactly what to study every day.

## Requirements

* Subject input
* Exam date input
* Priority input
* Preparation level input
* Progress tracking
* Day-wise schedule generation
